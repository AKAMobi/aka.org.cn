打开显示服务器

  • 在任何Ｘ程序运行之前必须与显示器服务器（display server）建立一个连接

  • 服务器名：机器名＋显示器号＋屏幕号

    • hostname:display_number.screen
    • 在程序指定，否则由Display环境变量指定
    • 一个Ｘ服务器（Ｘ server）可以控制多个屏幕（终端），只要它们属于一个显示器
      Display *XOpenDisplay(name) /* open a display on the named server.
      　　 If successful then all the screens on
      　　　　　 the display may be used . */
      　　　char *name; /* e.g. nokia-bbs:0.0*/
      XCloseDisplay(display) /* close a display completely */

  • 当一个客户连接结束时，Ｘserver就释放这个连接的资源
