内容简介：
=========
本次讲座主要介绍当前 Linux 操作系统上的两种重要 Script 编程语言，
即 BASH 和 PERL。BASH 现在几乎是各种 Linux 发行版中 Shell 的事实上的
标准，掌握 BASH 可以帮助用户更为便捷、合理的理解、管理和使用 Linux
系统；虽然 PERL 是一种非常适合写 CGI 的语言，但是 PERL 的作用不仅仅如
此，掌握 PERL 如同找到了 Linux 编程的捷径。

主要内容包括:

BASH 部分
=========
1. BASH 简介
2. BASH 的基本语法
　2.1 变量
　2.2 表达式
　2.3 控制结构
　2.4 函数
3. BASH 中的特殊保留字
4. BASH 的其他高级问题

PERL 部分
=========
5. Perl 的特点、历史和许可证
6. 用 Perl 写 CGI 入门
7. Perl的变量
8. Perl的简单语法（与C语言语法的异同）
9. Perl的正则表达式使用
10.一些Perl的实际运用范列
