AKA 第二期 Linux 系列讲座预告

　　Linux 的普及乃大势所趋、势不可挡。Linux 环境下各种自由软件的技术思想、
文档乃至源代码都是开放的，但是对于我们已经习惯于在 Windows 环境下编程和工
作的程序员来说，入门仍然存在一些困难，诸如： 

　　　　对系统环境不熟悉， 
　　　　许多工具不知道如何去使用， 
　　　　看文档不知道该从哪里开始看起， 
　　　　不习惯 Linux 下 C 语言代码的编程习惯， 
　　　　不知道如何使用 Makefile 等等。 

　　那就让我们来学习他吧，向已经在某些领域熟悉的同学学习，向网上的大虾学
习，向周围的朋友学习。用我们的智慧掌握这些本来就是“开放”的知识，用我们
的双手把梦想变成代码，将 Linux 运用到大家的学习、工作中去。 

　　我们在第一期讲座的基础上，结合 Linux 学习的内在规律，制定了一个 Linux 
系列讲座计划。我们的讲座针对的对象主要是对Linux 怀有浓厚兴趣并且在 C 语
言和其他操作系统环境下已经有一定经验的同学，本期系列讲座将针对同学们不同
层次的学习要求，分为“Linux编程系列”、“Linux内核系列”和“Linux 使用系
列”（初步的内容安排见后）。我们力争以讲座为契机，让大家的智慧在“开放”
的 Linux 平台下迸发出更多的火花，使国内出现更多的自由软件项目。

    关于第二期系列讲座的具体时间、地点我们将在8月底左右公布，敬请大家关
注。讲座的形式全部为免费讲座，自由听讲。另外本期系列讲座除了在内容上有所
提高和更加丰富以外，我们还打算成立技术支持组、项目支持组，实现网上报名与
考试等工作，希望这些能够对广大 Linux 爱好者有所帮助。我们欢迎所有同学来
听自己感兴趣的内容，同时关于这次讲座，如果您有任何意见和建议，请给我们来
信：                                                     aka@aka.org.cn

    同时我们希望所有对后面附录中所列的讲座内容比较熟悉的朋友能够与我们联
系，协助我们共同把 Linux 及自由软件的普及工作做好。从第一期系列讲座开始，
我们对每次讲座的主讲老师就有一定劳动报酬，不过我们更需要的还是一种自由、
协作、创造的奉献精神。如果你愿意与我们携手共建，请给我们来信：          
                                                         aka@aka.org.cn

    本次系列讲座将由 AKA 信息技术协会、蓝点软件北京研发中心、清华大学计算
机系科协、GNU/Linux研发中心及中科院计算所等多家单位共同协作组织实施。
