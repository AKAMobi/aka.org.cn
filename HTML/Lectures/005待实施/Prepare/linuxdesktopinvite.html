==> AKA “Linux桌面环境系列讲座”专栏 <==

自由、协作、创造－为了明天

　　AKA 致力于信息技术的传播，促进技术人员之间的交流与协作，AKA 讲座正是 AKA 这一工作目标的具体体现。我们的讲座是完全开放、免费的，技术资料是共享的。本次以“Linux桌面环境”为主题的系列讲座将主要致力于Linux桌面操作系统的普及。

　　也许你已经听说过 Linux 是一个操作系统，而且将会成为未来的主流操作系统，因此我们应该赶快了解它，才不至于在这个飞速发展的社会中落伍。

　　Linux以其开放的特点迅速风靡我们生活的这个小小星球，在高端服务器领域和低端的嵌入式设备领域它早已成为重要的系统平台，在日常应用的桌面领域 Linux 通过近两年的努力也获得了很快的进展，虽然与 Windows 相比还有差距，但是其功能已经可以基本满足我们日常学习和工作的需要。从安全和产业发展的角度考虑，我国政府正在大力倡导使用 Linux；另外Linux 不仅是正版软件，而且价格便宜（一般零售价不超过100元），因此未来的企业办公和家庭也必将会逐渐普遍使用 Linux。

　　AKA 希望紧扣时代脉搏，推动 Linux 的发展，宣传和普及 Linux ，希望 Linux 能在社会生活的方方面面生根发芽。在此我们希望热心 Linux 发展的朋友与我们积极联系，共同举办好“Linux桌面环境”技术讲座。

　　如果您有任何关于讲座的意见和建议，请一定给我们来信：aka@aka.org.cn。
