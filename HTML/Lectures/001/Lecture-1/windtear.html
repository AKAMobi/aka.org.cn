1.linux 与 windows 的比较
感觉较深,有时候对微软的做法 深恶痛绝，可又没什么办法, 只好任其宰割（不要误解，我肯定不买微软的东东的:PPP)当我装上了 bluepoint 之后，我进 windows 的时间已经不到 50% 了，（有时，是别人用我的机子，只好让他们用 windows)唯一遗憾的是，我要用 autocad 所以还不能把 windows 格掉

2.对 man 的用法更熟悉了
具体结构清楚了，以前不知道那么多 man 都是干什么的也没有看相关的文档，所以是个悬案(//shy)

3.c 语言的变成风格是我产生了震动
以前在 windows 下编程, 有时能缩进好几级现在一看，缩进三级都得考虑修改算法了, 原来禁止三级市从这儿来的:))

4.对 vi 的用法得到了复习和提高
以前看过 vi 的实用大全，可惜有些东西都不用 只有 i I a A o O x dd R 等用得多一些别的就用的很少，应该说不知道，没有在意看来还得好好看看相关文档了这么强大工具，如果不好好用，简直对不住老一代革命家

5.编译的知识
以前会用 gcc -o 别的参数就很少考虑了，也没太在意那些选项

6.调试的知识
是不是可以说 debug 很烂，（不过 dos 下还不错)

7.那几个小工具真不错 strip 的演示给我指明了方向:)

8.make 和 Makefile
虽然以前经常用，也看了不少别人的 Makefile (e.g. firebird bbsnet xdict 等等很多) 可都是为了某种特定目的而看的看看需要的，就不理会（不在意）别的了 所以还不是很清楚，这次听了，革命有了旗帜和对象:PP

9.其它工具 tar 等是经常用的了 (tar zxvf (zcvf) 用得多一些 find ./ -name ?? -print 也用的不少 cmp diff 用过，ed emacs 就用的很少了.听了讲座说 emacs 那么牛，（以前也怀疑过应该很牛) 就是没有直观认识，也没亲自操作过 （好好学习，天天向上）
