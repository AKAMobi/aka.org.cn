2000年 AKA 新闻回顾

（1999 - 2000）

1、AKA 筹办第一期 Linux 讲座

　　从2000年初开始，在蓝点软件公司以及GNU/Linux研发中心的协助下，为推动高校及高科技企业对Linux技术的深入研究，AKA精心组织筹划了第一期 Linux 编程系列讲座。由于本期讲座准确的将讲座目标定位在帮助有一定基础的Windows编程人员迅速熟悉Linux 编程上，加上讲座有内容，有深度，而且完全免费、公开，因此引起了许多在校学生的强烈反响，也获得了良好的社会影响。

　　第一期系列讲座持续一个学期，到2000年7月初结束，共举行了十一期讲座，内容涉及编程、内核和一些高级应用，最后 AKA 还向十多名勤奋学习 Linux 知识的学员赠送了各种 Linux 发行版光盘。根据我们的不完全统计，总共约有800多人次参加了讲座，通过网络报名的朋友有200多人，在我们的签到表上有统计的有近100人，其中部分 Linux 爱好者还坚持听完了所有的讲座，这种精神着实让我们非常感动。虽然由于 AKA 是第一次举办这样的讲座，在讲座的内容和组织上都还存在不少有待改进之处，但是我们欣慰的看到讲座得到了广大同学和 Linux 爱好者的认同，得到了许多热心朋友的帮助，对 Linux 在高校中的深入普及起到了重要的推动作用。

2、AKA 筹办第二期 Linux 讲座

　　从2000年8月开始，AKA 用近半年的时间筹备和举办了第二期 Linux 讲座。在第一期讲座的基础上，第二期讲座在内容和课程质量上都做了进一步的努力，以便能将更多、更好的知识普及给大家，为国内的软件人员多注入一些能量。第二期讲座在内容上做了扩充，整个讲座分为编程、内核、应用三个系列，总共24次讲座，分别面向不同层次、不同需求听众的需要。

　　第二期讲座同样受到了广大学子、Linux爱好者、研发人员和其他社会各界的好评。根据我们的不完全统计，共有231名网友报名听讲（讲座正式开始后还有陆续的报名未列入统计范围内），总计有1300多人次前来听讲。本次系列讲座同样得到了蓝点软件公司的大力赞助，以及集群网络公司的大力赞助，在此向他们表示感谢。同时讲座举办过程中也得到了清华大学计算机系科协、GNU/Linux研发中心、中科院计算所的大力协助，此外还得到了众多Linux爱好者和老朋友的支持与帮助。在此一并表示感谢。

　　通过两次讲座，更多的人已经吸收了 Linux 这个自由奔放系统的一些高级知识。我们相信这个星星之火，一定可以燎原。

3、AKA 新成员见面会、老成员交流沙龙

　　10月15 日上午，AKA 核心成员在盒子酒吧举行沙龙，讨论下一阶段的工作计划和安排，与会的 AKA 十多人，大家进行了愉快而卓有成效的交流。12月3日，AKA 主要成员又举行了见面会，商讨了一些 AKA下一阶段工作开展等问题。

　　11月25日，AKA 新成员沙龙 如期在盒子酒吧举行，新老AKAer聚集一堂，共商 AKA 未来发展。

4、第四期 AKA 杂志出笼

　　12月末，AKA 第四期杂志《互联网的历史和知识》与广大读者见面。以后 AKA 还将继续将沙龙举办下去，向国内广大信息技术从业人员和学生介绍新知识，新技术。

　　配合杂志的出版，AKA 还筹划建立了杂志副刊，专门介绍当前最新的技术热点。

5、招新工作

　　为了推动 AKA 各项工作的开展，进一步实现自由、协作、创造的宗旨，2000年下半年 AKA 协会筹办了两次面向公众的招新工作，为 AKA 的持续稳定、健康发展打下了良好的基础。

6、各种沙龙、聚会

　　11月5日，8名AKAer在一起围绕网络安全话题进行了热烈的讨论，参与者踊跃发言，交流各自的知识和见闻，整个沙龙是有益和成功的。

　　11月26日，AKA 举行 OS 及嵌入式操作系统首次主题沙龙，xuas主持了第一次沙龙的讨论工作，许多朋友作了即兴发言，介绍了自己的研发工作，大家还讨论了沙龙今后的运作方式。

　　12月17日，操作系统小组又在北大举行了一次关于嵌入式系统方面的主题讨论。

　　12月24日，AKA 诚邀各路软工高手举行了一次软工方面的沙龙，探讨了关于软件研过程模型、XP 等方面的问题。与会的新老朋友集聚一堂，互相交换了自己在软件工程方面的经验和想法。

7、AKA 协助国家863高性能计算机专家组组建 共创软件联盟

　　1月，国家863计划将支持下创办了共创软件联盟，AKA参与了联盟的创建工作，并帮助联盟最初起步做了部分宣传工作。 并于11月20日在 共创软件联盟主办的 2000 年中国自由软件发展及应用战略研讨会――暨第一届中国自由软件发展及应用论坛上作了关于组织建设方面的发言。

8、AKA 主页不断更新

　　2000年秋，AKA 注册了自己的永久域名：www.aka.org.cn 。同时 AKA 网站还建立了ChinaNet中的镜像，以给教育网外的用户更快的访问速度。网站本身也进一步充实、完善了站内的内容，美化了站点的外观，建立了站内信息搜索，建立了三个邮件列表，方便AKA 成员之间的交流 。截至2000年11月29日，AKA 主页首页访问量突破了 10 万次。

9、AKA 2000年的各种讲座

　　除了前面提到的两期 Linux 系列讲座外，AKA 还举办了各种其他技术讲座。如：

　　9月27日的自由软件项目LVS(Linux Virtual Server)技术讲座，项目主持人张文嵩先生远道从长沙来北京亲自给大家做了该项目的深入分析。

　　11月18日，AKA 邀请古靖雨先生在北邮给广大 Linux 爱好者讲授了“Linux 黑客之道”。

10、其他工作

　　AKA 推出新的自由软件项目：MiniGUI、MiniTerm、GIRL、Diablo II closed Server。

　　另外为了配合 Linux 讲座的进行，AKA 还组织人力、物力为 Linux 爱好者组织影印了《Beginning Linux Programming》一书；总结了 Linux 内核研究推荐书目列表。
