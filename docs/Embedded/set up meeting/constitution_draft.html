AKA嵌入式开发兴趣小组章程（草案）
AKA嵌入式开发兴趣小组是AKA的分支机构，秉持和实践AKA“自由、协作、创造”的崇高理念，坚持AKA的非赢利原则。

AKA嵌入式开发兴趣小组以人才和技术为根本，以正规的组织化开展工作，致力于建设成为一个进步的、强有力的、开放的、公益的技术团体。

AKA嵌入式开发兴趣小组为全体成员所有，为小组成员提升技术水平和实现价值提供并创造机会。

AKA嵌入式开发兴趣小组的整体根本利益不容侵犯。

AKA嵌入式开发兴趣小组的执行机构是小组工作组。小组工作组在小组常务组的领导下开展工作。小组常务组必须体现全体成员的意志，集中全体成员的智慧，接受全体成员的监督，确保组织的有序运作，确保组织正确的发展方向。每届常务组任期两年。

AKA嵌入式开发兴趣小组的根本组织原则是民主集中制原则。

AKA嵌入式开发兴趣小组的根本人事制度原则是奖惩分明，任人唯贤原则。

AKA嵌入式开发兴趣小组的根本财务制度原则是公开、透明原则。

AKA嵌入式开发兴趣小组成员管理的根本制度是会员制。

AKA嵌入式开发兴趣小组技术管理的根本制度是项目负责人制和文档化管理制度

AKA嵌入式开发兴趣小组的所有成员应有顾全大局、维护团结、反对分裂的自觉性。组织内部的任何矛盾应在组织的框架内寻求解决。
