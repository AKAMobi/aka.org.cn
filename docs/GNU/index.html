自由软件文化

　　“自由软件”　实际上指的是一种自由，而不是价格。
　　“自由软件”　中的自由是指用户运行、拷贝、研究、改进软件的自由。
　　“自由软件”　使用“copyleft”来合法地保护每个人的自由。

　　想更多的了解“自由软件”吗，请阅读我们自己翻译的“自由软件”哲学论文。

MiniGUI

下载MiniGUI

　　MiniGUI 是在 Linux 控制台上运行的多窗口图形用户界面支持系统，可以在未来以 Linux 为基础的应用平台上提供一个简单可行的 GUI 支持系统。

MiniTerm

下载MiniTerm

 　　MiniTerm 是个专门用来上 BBS 的 telnet 客户端软件，它的主要想法来源于 M$ Windows下的 CTerm 和STerm ，提供了诸如自动登录、防止发呆、自动消息回复、鼠标支持等方便的功能。
　　MiniTerm是个以 GPL 版权(请看COPYING文件)发布的自由软件( Free Software )，提供源代码，如果你愿意，可以自己修改并添加新的功能。
BigEar 项目 　　Big Ear(大耳朵)是一个语音软件，可以朗读任意汉字文件。这是一种Text To Speach系统。 大耳朵可以帮助盲人朋友阅读大量的电子读物。也可用于校对。

网络蝗虫 项目 　　Netlocusts(网络蝗虫)是一个基于Linux操作系统、运行于X窗口下的、可断点续传的多线程下载工具。与Windows下的同类软件相比，具有下载快、可控性好等特点，而且目前在Linux下还没有流行的同类产品，可谓独树一帜，具有广大的需求潜力和广阔的市场前景。 > > >
GIRL计划 　　GIRL是图形界面表述语言（Graphic Interface Represent Language） 的缩写，一个新的界面软件开发工具包。主要设计思想是采用类似Motif的UIL形式，以独立文本的形式构造软件界面。应用程序开发者只要在GIRL文件中给出界面描述，就可以把复杂的界面元素作为整体应用。
Script C 项目 Script C计划被设计成一种解释执行的、C语言风格的脚本语言。它用来完成bat批处理无法完成，而用C语言写程序又太复杂了的这类工作。它没有java的高安全性但能够直接使用系统的许多资源，可以使用API，在功能上要比java强大。目前只打算支持 Windows 系统，包括 95/98/SE/ME/2000/NT/2000/XP 。 > > >
