帮助保护编写非自由软件和自由软件的权利

(jpeg 7k) (jpeg 141k) 因专利权问题而没有gif图

编写非自由软件和自由软件的权利正受到“软件专利”和“观感相似性”界面版权法案（“look-and-feel” interface copyright lawsuits）的威胁．

自由软件基金会（Free Software Foundation）通过各种途径与之斗争，其中就包括支持和加入“自由编程协会”（League for Programming Freedom）。

该协会是一民间组织，由教授，学生，经商者，程序员，用户和软件公司组成，献身于重新得到编写程序的自由．该协会并不反对议会所希望的法律系统――个体程序的版权。该协会的目标是扭转由特殊利益引起的判决所造成的变化（有关软件的版权）。

自由软件基金会希望您尽快加入我们来与这些威胁斗争。两个好方法是加入和/或帮助该协会．

该协会与自由软件基金会无关，且与自由软件的观念无关。自由软件基金会支持该协会，因为如同任何小于微软的软件开发者一样，它正受到软件专利和界面版权的威胁。您也一样受到它们的威胁。在您或您的雇主遭起诉之前，很容易忽视这个问题，但未雨绸缪更加谨慎。
