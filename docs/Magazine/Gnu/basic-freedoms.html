自由软件基金会（FSF）支持Internet上的言论、出版、交流的自由，请看：

  • 最近，美国最高法院将会宣布关于通讯合法化行动(Communications Decency Act，CDA)的合法性挑战的最终决定。请访问the Citizens Internet Empowerment Coalition（公民Internet权益联盟）以获取此案例的最新消息。
  • Voters Telecommunications Watch（选举者电信观察）和他们优秀的电子邮件组公告。
  • 检查GNU Emacs 描述了CDA是多么需要GNU 项目来检查GNU Emacs――和这种行为是多么荒谬地得到检查人员所期望的相反的结果。
  • F.A.C.T.Net Inc.是一家非盈利性Internet文摘、新闻服务、图书馆、对话中心和档案中心，专门从事国际自由思想、自由演讲和私人权利的促进和保护。
  • 网上言论、出版、交流自由的蓝丝带行动（Blue Ribbon Campaign）
  • 您可以阅读1996年6月上诉法院关于拒绝Internet审查制度的决定，但请记住，这并 不是 最终的决定，首先，最高法院将会同意或者否决；其次，国会有机会去寻找另外一种审查的模式。
