1999 Apr. 29

knoqing将他的一些程序的源码公开，欢迎前去访问！

ftp://166.111.53.221/OpenSource/Kongming
别人的 Open Source 也可以放在单独设置的个人目录中。

放上去的程序有：
MiniGUI4DOS: 完整的 MiniGUI for DOS 源文件，包括一个例子程序。
简介：一个 DOS 下的单窗口GUI系统

MiniGUI4Linux: MiniGUI for Linux 的源文件，当前正在开发中，
但有一个演示程序可用。
简介：这是个Linux 的简单多窗口系统现在已经完成了多窗口的
管理功能，其中图形设备的管理，窗口图形输出的剪切是非常
关键的功能，这些大部分均以完成了。接下来的工作主要是开发
一些诸如菜单，控件等基本的GUI元素。大概尚需要2到3个月的
时间。

Sheet4Win32: 用于表格处理的 MFC 派生类。有一个示例程序，文档不齐备。

另外，在http://166.111.53.221/LinuxNNN/contents.html
上有他最近编译的有关 Linux 内核的工作，也可以访问
