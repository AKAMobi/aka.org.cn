    与Microsoft Windows 98或NT不同，Linux操作系统并不是一个单一的产品，在市场上有多种发行版本，它们并不是都一样。

    所有的发行版本具有一样的linux内核，内核包含着所有核心的操作系统功能以及网络堆栈。另外，他们都提供标准的工具popular system libraries，一系列的应用程序，一些打印的文档，以及有限的技术支持。每种发行版都带有X Window 系统以及一个图形用户界面，Web server，e-mail server，以及FTP server。真正不同的地方在于其安装，配置，支持，以及第三方应用。

    例如，安装过程可能是费劲的，在设置象网卡这样的硬件时用户可能会陷入泥潭。多数Linux网络配置程序都有一个另人难忘的命令行界面，并且需要用户具备专业知识。而技术支持——至少是正式的技术支持，是有限的。而且，虽然Linux发行版中带有许许多多的应用程序，但是没有一个象Microsoft Office这样成熟易用。不过，如果你愿意投入时间去琢磨它，最终还是可以将一切配置好的。如果你不想深入研究Linux，Red Hat应该是你最好的选择，因为它带有最成熟的安装程序。Caldera的安装程序无法与之相比，但是它带有一些吸引人的商业应用，可以适应中小型组织的需要。

    为了从Windows 用户的视角来评价哪种Linux发行版是最好的，我们测试了Caldera OpenLinux 1.3，Debian GUN/Linux 2.0，Linux Slackware 3.6，以及正式的Red Hat Linux 5.2。如果你想获得完整的Linux发行版列表，可以访问这个网址：www.linuxhq.com/dist-index.html

    这些接受评测的Linux发行版都有CD-ROM出售，价格在40到60美圆，如果从网上下载则是免费的。虽然每个发行版都带有文档，但是最好的Linux技术支持资源就是Internet，你可以从这里开始：Linux Documentation Project (metalab.unc.edu/LDP)。

Caldera OpenLinux 1.3


    一般认为Linux不是一个适合公司集团使用的操作系统，但是Caldera正试图改变这一状况。Caldera OpenLinux(零售价59美圆)是唯一带有全面商业应用的发行版，其中包含了它自己的NetWare客户端软件，一个数据库服务器(Sybase ASE)，以及完整的商务套件。

    与Red Hat类似，Caldera也提供了启动盘，可以简化安装。总的来说，Caldera的安装不如Red Hat那样图形化和直观，不过，除了需要使用Fdisk进行分区之外，安装Caldera的安装似乎也不是很难，而且也不需要用户太多的干预。Caldera包括了Red Hat Package Manager(RPM)，使得在Linux下安装和反安装应用程序变得容易——至少在Linux的标准上是如此。LISA(Linux Instalation System Administration)是一个图形化的工具，用于配置基本的网络客户端参数以及其它一些特性。在实际应用中，我们还是需要化不少时间手动编辑配置文件。

    Caldera最强的优势之一是KDE(Kool Desktop Environment)，这是一个图形化的桌面环境，使Linux在外观和操作上与Windows类似。而且还带有由Star Division开发的StarOffice 4.0，这是一个与Microsoft Office类似的套装软件，这样你就有了可适于中小企业的完整解决方案了。最后，Caldera还提供多种形式的技术支持方式，包括电话和e-mail支持。

    Caldera OpenLinux 1.3，零售价：59美圆(双CD，Linux Kernel 2.0.36，启动盘，开始指南，30天技术支持。)。系统需求：386或更好的PC，16MB内存，350MB磁盘空间(如果安装额外的应用需要450MB)。www.calderasystems.com ； ftp.caldera.com/pub

Debian GNU/Linux 2.0


    Debian GNU/Linux 2.0(零售价38.95美圆)在主要的发行版中是唯一仍由一群志愿的程序员出品的。该版本以及其许多强有力的特性，使之成为黑客中流行的选择。但是Windows用户对此要考虑清楚。

    Debian被我们选为最难安装的发行版。该系统没有启动盘，不过你可以自己制作一张，或者如果你的系统允许，你也可以从CD启动。安装程序带有Fdisk用以处理分区问题，这个安装程序部分上是图形化的，但是仍然不容易使用。如果你对你机器的每个硬件都很了解而且知道如何配置网络，这种类似于向导的安装过程还可以接受。在安装完成后，我们发觉系统中几乎没有任何应用程序。emacs文本编辑器以及Web浏览器必须分别用Debian的dselect程序安装，dselect程序向来以难以使用而闻名。该公司称，在新的Debain GNU/Linux 2.1中将包含新的应用安装程序。

    当你完成安装后，Debian包含的X Window可以使工作稍微容易一些。该发行版最大的买点在于包括一个可选的包管理器——Debian Package Management System，它可以在安装之前对新的应用程序进行扫描，并且检查系统现有的配置情况，以决定所需安装的包，以免发生冲突。Debian由Linux Press发行，带有一本268页的手册。


    Debian GNU/Linux 2.0；直销价格：38.95美圆(包括三CD，Linux kernek 2.0.35；Debian Linux 用户指南第二版；30天的e-mail技术支持)。系统需求：386或更好的PC，16MB内存(为支持图形界面)，100MB硬盘空间(推荐200MB)。由Linux Press发行， www.linuxpress.com ， ftp.debian.org 。

Linux Slackware 3.6


    这曾是最流行的Linux发布版，由Patrick Volkerding's software制作，不过其市场领导地位现在已经让位于制作更精细的Red Hat了，可以它在老牌Linux用户中还是最流行的。与Debian一样，Linux Slackware 3.6(直销价格39.95美圆)没有启动盘。当你从映像中制作启动盘时，众多的硬件选项另人生畏。我们最后发现了一个通用的磁盘映像(bare.i)，它总是可以工作的不错。

    Slackware 不如Caldera或者Red Hat那样易于使用，但是还是有些优点的。如果我们在安装过程中忘记将某些设置写入config script就重启动了，Slackware将自动向root用户发一个e-mail，说明问题所在以及该如何恢复。它还提供了一批不错的configuration scripts，可以自动完成一系列的命令。

    Slackware最好的特性是在其CD上有一套已经安装好的Linux和X Window，如果你的系统支持从CD启动，你可以直接试用Linux而无须往硬盘上安装任何东西。Slackware带有一个转换工具可以管理RPM包以及一个窗口管理器FVWM95。Slackware原本由 Walnu Creek CDROM发行，不过在我们这篇文章快完成时，Volkerding建立了一个正式的Slackware Web站点。


    Linux Slackware 3.6。直销价格：39.95美圆(包括4CD，Linux kernel 2.0.36，安装指南，技术支持)。系统需求：386或更好的PC，8MB内存，200MB硬盘(完全安装需要500MB)。由Walnt Creek CDROM发行， www.slackware.com ； ftp.cdrom.com/pub/linux/slackware


Official Red Hat Linux 5.2


    Official Red Hat Linux 5.2(零售价49.95美圆)已经成为Linux市场中最重要的一员，这主要是因为它提供了最优秀的安装程序以及先进的包管理程序。如果你是第一次使用Linux，Red Hat是最好的选择。

    象Caldera一样，Red Hat包含了一张启动盘。安装程序检查你的硬件，并且给出合乎逻辑的安装建议(笔记本电脑的用户一定会对其PC Card支持能力感到满意)。Red Hat还提供了WorkStation和Server两种预设的安装模式，这些模式下，分区以及应用程序的选择都是设置好的，当然，你也可以选择用户安装方式以自行决定分区方式。在分区方面，Red Hat提供了一个叫Disk Druid的应用程序，这个图形化的安装程序要优于Fdisk——不过你还是需要对分区和目录结构的基本概念有个了解。

    为了使配置工作更容易，Red Hat包含了一系列的工具：用于设置X Window系统的Xconfigurator以及XF86Setup，前者是一个向导型的培配置程序。Linuxconf使配置网络变得更容易，包括配置PPP客户端，DHCP，以及DNS。(不幸的是，Linuxconf仍在不断的改进中，因此缺少文档)，更好的是，如果你在安装过程中使用Red Hat的boot disk creation script，它将存储你的网络以及XF86Config信息，这样你就不需要一次次的反复输入这些信息了。

    Red Hat因其包管理程序(RPM)闻名，这是一个开放源码的程序，在其它许多发行版中也广泛应用。RPM使安装和反安装应用变得安全，避免了程序冲突，甚至可以对内核本身进行升级而无须重装整个系统。

    Official Red Hat Linux 5.2。零售价：49.95美圆(包括3CD，Linux kernel 2.0.36，启动盘，安装指南，90天的技术支持)。系统需求：386或更好的PC，16MB内存，120MB硬盘空间(Workstation方式需要450MB，Server方式需要1.6GB)。Red Hat Software， www.redhat.com, ftp.redhat.com/pub
