AKA主页\AKA沙龙

　　AKA 的理念是 自由、协作、创造，营造一个交流的环境是我们重要工作之一。AKA沙龙就是给大家一个面对面的交流机会，我们希望在每一次沙龙中大家可以从交流中获得新的信息，在别人的点拨下解决一些疑难问题，在热烈的讨论中迸发出智慧的火花。

　　如果你有好的话题，欢迎给我们来信：aka@aka.org.cn。 我们将根据你的建议以及其他 AKAer 的感兴趣程度，组织大家一起讨论。

　　参加（或希望参加） AKA沙龙活动 的朋友，请务必阅读一下 沙龙 FAQ (Ver 1.3) 。
