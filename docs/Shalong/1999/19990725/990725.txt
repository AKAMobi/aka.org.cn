    周末沙龙总结

    1999年7月25日（周日）上午9:00，20余名AKA主要成员和一些Linux的热心支
持者参加了这次以“LisoLeg、Linux内核剖析 和 Linux Virtual Server”为主
题的沙龙活动。本次讨论技术性强、涉及技术新、参与者发言和提问踊跃，整个
沙龙是非常热烈和成功的。

    沙龙内容分两大部分：
    <1> 由老铁介绍lisoleg的工作进展；
    <2> 由来自长沙的张文松介绍了他做的“Linux Vitual Server Project”

    老铁首先简单介绍了Lisoleg, 即Linux源码学习小组，的基本情况；接着，
他介绍了Lisoleg的主要工作内容；然后还介绍了Linux源码的整体结构，进程
调度、内存管理、虚拟文件系统、网络接口、进程间通信；最后，老铁从软件
工程的谈了Linux开发的特点以及为什么能获得成功的原因。关于lao_tie的讲
座更详细的内容可以从AKA WWW网页中(老铁提供的材料)得到。

    张文松介绍了他的"Linux 虚拟服务器"，同样引起了大家的浓厚兴趣。虚
拟服务器成本低廉，可伸缩性好，安全性能好等优点，具有很高的实际应用价
值。该方案已经为美国多个ISP站点采用，并获得好评，实际应用表明集群的规
模可以达到254台！而且该技术还将在Linux Kernel 2.4.0中作为新特性被加入。
张文松详细向大家介绍了他的设计思想和结构，并在演讲结束后进行了自由讨论，
大家就此方案进行了热烈的讨论，张文松并解答了大家的各种问题。关于该内容
的具体技术细节请看AKA WWW 主页（张文松提供的材料）。

    * 欢迎大家访问laotie他们的Lisoleg的主页：
                           http://lisoleg.yeah.net
    * 欢迎大家访问Linux 虚拟服务器的站点：
                           http://proxy.iinchina.net/~wensong/ippfvs

    * 最后再次感谢walt教授为我们提供会议场所，并积极指导了我们的讨论。

    这次沙龙对推动AKA发展，激发AKA小组热情和创造力起到了极大的作用。
