内容：

    WebSphere 是因特网的基础架构软件，也就是我们所说的中间件。它使企业能够开发、部署和集成新一代电子商务应用（如 B2B 的电子交易），并且支持从简单的 Web 发布到企业级事务处理的商务应用。WebSphere 转变了企业对客户、合作伙伴及雇员之间关系的管理方式。
    WebSphere 提供了一个业界领先的，用于开发、部署和维护基于 Web 应用的，且遵循 J2EE 规范的软件平台。该平台包含了可跨越最广泛平台（最新的统计超过 35 个）的主要的开放标准技术（包括最近的 Web Services 规范）的最深层实现。有了 WebSphere，IBM 将继续其对领导创建新的开放标准的承诺，并且既遵从各种如 Linux 的开放式技术平台，也遵从诸如 J2EE、XML 和 UDDI 这样的现有标准。这些努力为 WebSphere 客户提供了更短的开发周期和更可靠的，更强大的互操作性的电子商务应用程序 － 从简单的 Web Services 到复杂的基于事务的解决方案。
