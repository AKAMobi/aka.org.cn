Localization at SGI

How to Localize

Localization Software Process

The main tasks in the localization process are listed below in approximate chronological order, although they tend to overlap. In addition to coordinating these tasks, localization project management involves scheduling, obtaining estimates, and entering into agreements with outside vendors.

Creation of glossaries for each language and product (outsourced, reviewed in-house)

Pre-localization testing of multibyte characters and different locales (in-house)

Translation of user interface strings and messages
(outsourced)

Translation of documentation
(outsourced)

Building/testing/debugging
(part outsourced, part in-house)

Software

Online documentation
(includes testing of documentation against software)

Preparation of images for final testing
(in-house with product group)

Final QA/validation
(joint effort between localization, the product group, and
QA/validators in different subsidiaries)
