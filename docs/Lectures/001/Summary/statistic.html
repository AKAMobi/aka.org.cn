AKA“Linux 编程”系列讲座的 统计分析 及 学员奖励办法
　　======================================================

　　本次讲座得到了众多 Linux 爱好者的支持与爱护，根据我们的不完全统计，总
共约有800多人次参加了讲座，通过网络报名的朋友有200多人，在我们的签到表上
有统计的有近100人，其中部分 Linux 爱好者还坚持听完了所有的讲座，这种精神
着实让我们非常感动。

　　本次来听讲座的朋友以高校的学生和各科研机构的科研人员居多，大约占总人
数的90％，其他还有部分公司企业中的关心前沿技术的技术人员。这个结果显示了
近期 Linux 编程技术普及的对象还是以高校和科研机构为主，不过同时也开始受到
产业界自发的注意。

　　来参加讲座的朋友几乎都是20岁到30岁之间的年轻人，显示了Linux技术是一个
新兴的技术，当前还主要在年轻技术群体中流行，同时也说明了Linux发展的巨大潜
力。另外在性别比例上，男女比例大约在9:1左右，一方面说明了对 Linux 技术比较
感兴趣的还是以男性为主，另一方面也向大家传递了这样一个信息，就是巾帼并不让
须眉。

　　通过大家对 Linux 编程讲座的关心程度和参与程度，我们不难看出，Linux操作
系统和技术的普及在高校中已经形成了一定的规模，并且根据我们的预测，还有不断
上升的趋势。而且特别让我们感到欣慰的是，在当前 Linux 社区整体比较“热”的
情况下，国内的高校的学生以及其他已经跨出校园的年轻人已经开始不仅仅满足于安
装、使用 Linux 了，而是开始积极的参与到比较深入的 Linux 技术的学习和研究之
中。这恰恰预示着我们中国的程序员将开始掌握 Linux 的底层技术，并逐步对 Linux
的未来发展提出我们自己的看法，展示我们的力量。希望我们在不远的将来能够看到
一大批由中国程序员创造出来或参与的自由软件。

　　为了表达我们对大家的谢意，特地准备了一些小奖品，送给那些积极参与讲座
的朋友。
