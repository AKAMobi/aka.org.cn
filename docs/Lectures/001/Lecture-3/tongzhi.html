第三次讲座通知：

　　主题：自由软件项目介绍(LIDS、SmartBootManager、MiniGUI)
　　时间：4月23日（星期日）下午 2:00 － 5:00
　　地点：清华大学西主楼 2 区 404 （从西主楼二区上四楼， 一直右转，见到“A管理模式培训中心”即是）。
　　主讲：魏永明，苏哲、谢华刚

　　内容简介：
　　* LIDS = Linux Intrusion Detection System，是对 Linux 核心的一个patch ,用来增强 Linux 核心的安全性。LIDS的开发者将亲自向你描速如何用LIDS构造一个安全的Linux系统； (http://www.lids.org)
　　* SmartBootManager是suzhe利用他丰富的计算机内部结构和机 理方面的知识，利用业余时间开发的一个可以与PQMagic相媲美 的分区工具。(http://suzhe.home.chinaren.net)
　　* MiniGUI是Kongming面向嵌入式系统应用开发的一套小图形操 作函数库，可以应用在手持设备、嵌入式系统和工控系统等应用 中。(http://ymwei.home.chinaren.net)

　　讲座形式：三个项目各20分钟的简单介绍 ＋ 小组讨论

　　备注：自由软件的一个重要思想是市集式的开发，一个好的项目 需要众人的参与才有生命力，本次讲座将给大家一个直接与系统 开发者交流的机会，在经过简单介绍之后，将根据个人兴趣进行 分组讨论。因此我们更换了地点（请大家注意），希望曾经报名 并领到票的同学能够按时前来，如果你有事不能前来请给我们发 信，我们将把你的听课资格顺延给其他报名的同学。根据我们前 两次的统计，对于累计两次不能在周末来听讲座的同学我们将取 消他的资格，将这个机会让给其他朋友，同时对于能够全部出勤 的同学我们将在本次活动结束的时候给予奖励。另外在场地情况 许可的情况下，AKA也非常欢迎所有对上面三个项目感兴趣的朋友 前来交流，谈你的想法，提你的建议，促进交流与合作。
