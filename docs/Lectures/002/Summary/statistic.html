第二期 AKA“Linux 编程”系列讲座的 统计分析
　　=============================================

　　本次讲座得到了众多 Linux 爱好者的支持与爱护，根据我们的不完全统计，总
共约有1700多人次参加了讲座，通过网络报名的朋友有250多人，其中部分 Linux 爱
好者还坚持听完了所有的讲座，这种精神着实让我们非常感动。

　　本次来听讲座的朋友和第一期讲座相比还是以高校的学生和各科研机构的科研人
员居多，大约占总人数的75％，但是随着Liunx技术的普及和产业化，讲座愈来愈受到
企业的关注，许多公司的技术人员也积极参加了他们所感兴趣的讲座。这个结果显示了
近期 Linux 技术已经愈来愈受到产业界的注意，他们已经开始了将技术转化为产品的
革命，可以预见不久的将来将是一个 Linux-Anywhere 的世界。
