内容简介：
=========
本次讲座主要介绍的是linux核心中的动态内存分配算法，主讲者将从
一个普通程序员的角度去剖析这个有趣的话题。相信只要是有计算机
基础知识的同学即可听懂。讲座深入浅出，剖析核心，机会难得，欢
迎大家前来学习、探讨。

对于应用程序来说，内存的动态分配与释放使用两个库函数：malloc()
与 free()。而在Linux的核心，则使用的是另外两个底层核心函数
kmalloc() 与 kfree()。这两套函数有什么区别呢？本次讲座将从这两
个函数入手，讲述Linux核心内存分配器的实现算法。

动态地分配与释放内存对象是Linux核心中最普通的操作。因此，一个
快速的核心内存分配器对核心的性能的提高显得由为重要。本次讲座将
全面讲述Linux核心内存分配器的详细实现。包括页面级分配器，核心
内存分配器以及分页系统。
