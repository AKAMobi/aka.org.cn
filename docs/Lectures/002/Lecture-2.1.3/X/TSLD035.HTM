KDE简介

  • Linux上第一个真正的桌面

  • 开发的原因

    • 没有通用的Drag&Drop协议；
    • 没有基于对话框的桌面配置工具；
    • 没有一致的应用帮助系统；
    • 没有统一的应用开发平台；
    • 没有复合文档平台；
    • 在应用级没有网络透明性；
    • 在X下写程序非常乏味。

  • KDE的组成

    • KDE桌面环境(包括WM)；
    • KDE应用开发平台；
    • KDE办公套件。
