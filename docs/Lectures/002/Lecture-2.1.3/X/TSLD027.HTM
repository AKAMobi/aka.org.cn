GTK的控件

  • 使用一个控件的步骤

    • gtk_*_new 初始一个控件
    • 绑定控件与信号及事件
    • 把它包装到一个包容器中（gtk_container_add，gtk_box_pack_start）
    • 告诉GTK显示这个控件（gtk_widget_show）

  • 控件体系类似于MFC的类库，子类具有父类的特征

    • 在使用时要使用类型强制转换

  • 控件与窗口

    • 控件不一定有接收信号的窗口（GtkPixmap，GtkTable等）
    • 使用EventBox绑定一个窗口
