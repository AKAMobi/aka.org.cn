  • 选择（selection）

    • 属性（property），原子（atom）
    • ButtonPress

        • 声明为一个选择的拥有者，SelectionClear通知原主人
      • event.xbutton.button == Button2
        • 选择自己的目标原子属性

      • SelectionRequest
        • XChangeProperty生成被请求的选择的数据
        • XSendEvent发送给请求者
      • SelectionNotify,例子６
        • XGetWindowProperty获得数据并显示出来
