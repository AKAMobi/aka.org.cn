AKA 第二期 Linux 系列讲座报名开始了！

AKA 第二期 Linux 系列讲座的报名开始了，不好意思让大家多等待了几天，谢谢大家对这次活动的关注。

本次报名工作通过网络进行，具体链接为 http://register.aka.org.cn ，在大家开始报名前，请先仔细阅读下文。

第一：本次报名工作全部通过网络的WWW方式进行，们将不再处理在BBS上和通过Email报名的申请。
第二：本次系列讲座旨在普及Linux和自由软件的编程、应用和研究方面的知识，推动Linux、自由软件及开源软件的发展。所以系列讲座将是完全公开和开放的，任何人均可以免费去听讲，而且每次讲座结束后，讲座的所有资料也将会在AKA的网站上公开。
第三：想听讲座的朋友即使不进行报名和登记也完全可以自由的来听讲座，但是我们希望大家能够事先报名。
第四：为什么要报名？报名可以帮助我们分析哪些讲座更受大家欢迎，同时报名的朋友还可以有机会获得我们在讲座结束后为大家准备的奖品，另外报名才有资格参加我们最后的考试和认证以及其他我们现在还没有考虑到的许多好处。
第五：本次系列讲座专门设立了一个邮件列表，以便更方便、快捷的将每次讲座的预告和其他信息通知给大家。通过向： linuxStudying-subscribe@egroups.com 发一封任意标题和正文的信件即可加入我们的邮件列表，当然你也可以通过向： linuxStudying-unsubscribe@egroups.com 发一封任意标题和正文的信件来方便的退出该邮件列表。需要注意的是，263的免费邮箱好像不被该邮件列表接受。
第六：由于我们准备工作上的问题，第一期讲座将需要等到9月中旬举行。 其他工作：包括技术支持组、项目支持组以及讲座其他相关工作我们正在进行，希望大家能够关注我们的进一步通知。如果你有任何意见和建议，请一定来信告诉我们：aka@aka.org.cn
第七：本次报名活动到2000年10月15日截止，在2000年10月15日之前均可以通过我们的网站报名，在上述时间后将不再接受报名申请。

最后，我们非常希望我们的讲座能够推动国内 Linux 的发展，帮助和促使中国诞生出更多的自由软件项目和人才。

AKA
2000/9/3
