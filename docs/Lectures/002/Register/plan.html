AKA 第二期 Linux 系列讲座计划安排

本期Linux系列讲座将分为三大部分，分别为“Linux编程”系列、“Linux内核”系列 和“Linux应用”系列，详细安排预计如下：

Linux编程系列：（大约9月中下旬开始，地点清华大学）
Linux编程入门之C语言环境（gcc、gdb、Make、Diff、Patch）（魏永明）
Linux编程入门之脚本编程（Bash、Perl）（陶品）
Linux编程入门之桌面编程（X、QT、GTK）（李晓峰）
Linux编程提高之项目管理（CVS、automake/config、profile、程序优化）（耿岳）
Linux编程提高之文件系统及I/O（魏永明）
Linux编程提高之进程及进程间通讯（魏永明）
Linux编程提高之信号、终端及控制台（魏永明）
Linux编程提高之Socket编程（余津凯）
Linux编程提高之CORBA/GNOME （李晓峰）
Linux编程提高之I18N（国际化）（余明俭）

Linux应用系列：（9月17日开始，地点清华东门华业大厦五楼）
Linux的系统管理（用户管理、系统分区、/etc介绍、vim使用等）（耿岳）
Linux服务器配置方法，DNS、IP、网络等（李卓桓）
PHP、CGI网页设计和MySQL编程（廖阳）
服务器配置，FTP、sendmail、samba配置（Scaner）
Apache的配置及Emb-Perl编程（李卓桓）

Linux内核系列：（大约9月下旬开始，地点中科院计算所）
Linux的内存管理与缓冲机制（干戈）
Linux的网络部分的实现（干戈）
Linux的进程管理（卜东波）
Linux的文件系统（谢华刚）
Linux的中断机制与底半处理（张福新）
Linux的模块编程（向泓）
Linux的SMP（李晓峰）
