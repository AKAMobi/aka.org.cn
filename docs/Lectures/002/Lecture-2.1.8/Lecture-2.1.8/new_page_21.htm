如何从socket中读取一行语句？

对于很多基于telnet协议的服务，如FTP, POP3等，它们的操作是通过输入一行字符完成的，这就需要socket每次读取一行，这可以通过文件流的方式完成（详见上一条）。

程序如下：（设sockfd是已连接的连接描述符）

FILE *cin, *cout;

char *p, buf[1024];

 

cin = fdopen(sockfd, "r");

setbuf(cin, (char *)0);

}

fclose(cin);

close(sockfd);
