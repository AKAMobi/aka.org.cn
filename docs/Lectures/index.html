==> AKA “信息技术系列讲座”专栏 <==

自由、协作、创造－为了明天

　　AKA 致力于将最新、最有潜力的信息技术传播给中国的广大技术人员，促进技术人员之间的交流与协作，AKA 讲座正是 AKA 这一工作目标的具体体现。我们的讲座是完全开放、共享的，针对不同的技术领域，邀请该领域的专业人士来讲述当今技术潮流，勾画信息技术的未来，传授实用的技术方法。

　　如果你在某个领域还是一个新手，对于一些技术还不熟悉，还不知道从哪里入手，不知道到哪里去寻找资料……不用担心，在这里你就可以寻找到答案。让我们来努力学习吧，向已经在某些领域熟悉的专家学习，向网上的大虾学习，向周围的朋友学习。用我们的智慧掌握那些本来就是“开放”的知识，用我们的心灵来感受时代的脉搏，用我们的双手把梦想变成代码，将所学知识运用到大家的学习、工作的各个方面去。

　　AKA 在网上热心网友建议的基础上，结合不同时期的技术热点和重点，在我们的精心筹划和安排下，举办各种主题的技术讲座，让大家的智慧中迸发出更多的火花。

　　我们的讲座完全免费、开放，欢迎所有朋友来听你们自己感兴趣的内容，同时如果您有任何关于讲座的意见和建议，请一定给我们来信：aka@aka.org.cn。
