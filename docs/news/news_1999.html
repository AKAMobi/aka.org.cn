1999 Jul 25　

　　　　周末沙龙总结 1999年7月25日（周日）上午9:00，20余名AKA主要成员和一
　　些Linux的热心支 持者参加了这次以“LisoLeg、Linux内核剖析 和 Linux
　　Virtual Server”为主 题的沙龙活动。
　　　　本次讨论技术性强、涉及技术新、参与者发言和提问踊跃，整个沙龙是非
　　常热烈和成功的。 沙龙内容分两大部分： <1> 由老铁介绍lisoleg的工作进
　　展； <2> 由来自长沙的张文松介绍了他做的“Linux Vitual Server Project”

　　　　老铁首先简单介绍了Lisoleg, 即Linux源码学习小组，的基本情况；接着，
　　他介绍了Lisoleg的主要工作内容；然后还介绍了Linux源码的整体结构，进程
　　调度、内存管理、虚拟文件系统、网络接口、进程间通信；最后，老铁从软件
　　工程的谈了Linux开发的特点以及为什么能获得成功的原因。关于lao_tie的讲
　　座更详细的内容可以从老铁提供的材料得到。

　　　　张文松介绍了他的"Linux 虚拟服务器"，同样引起了大家的浓厚兴趣。虚
　　拟服务器成本低廉，可伸缩性好，安全性能好等优点，具有很高的实际应用价
　　值。该方案已经为美国多个ISP站点采用，并获得好评，实际应用表明集群的规
　　模可以达到254台！而且该技术还将在Linux Kernel 2.4.0中作为新特性被加入。
　　张文松详细向大家介绍了他的设计思想和结构，并在演讲结束后进行了自由讨论，
　　大家就此方案进行了热烈的讨论，张文松并解答了大家的各种问题。关于该内容
　　的具体技术细节请看张文松提供的材料。

　　　　* 欢迎大家访问laotie他们的Lisoleg的主页
　　　　* 欢迎大家访问Linux 虚拟服务器的站点
　　　　* 最后再次感谢walt教授为我们提供会议场所，并积极指导了我们的讨论。

　　这次沙龙对推动AKA发展，激发AKA小组热情和创造力起到了极大的作用。

1999 Jul 11　

　　周末沙龙总结

　　　　1999年7月11日（周日）上午9:00，18名AKA主要成员和一些AKA的热心支持者参
　　加了这次以“创业大赛 与 AKA发展”为主题的沙龙活动。尽管凌晨刚刚结束的女足
　　决赛给人带来了一些遗憾，但是AKA的讨论还是非常热烈和成功的。

　　　　这次沙龙的主要目的是为了讨论AKA的进一步发展方向，解决AKA当前急待解决的
　　问题。沙龙首先由原清华创业者协会会长yjf简单介绍了一下清华创业大赛的简单情况，
　　并且由此引发开来，yjf以清华创业大赛的创始人的亲身感受和经历，谈了他们在当初
　　创办清华创业协会时所走过的道路和遇到的困难和问题，以及是如何解决这些问题的。
　　他的亲身感受给AKA和参加沙龙的AKA成员带来了许多启发。接着现在正在开办公司的
　　lla以一个实际创业者的经历和大家畅谈了他的创业经历，同时lla阐述了自己对AKA的
　　许多设想和建议。然后到会的所有成员都很积极的谈了自己对AKA的看法和想法，为
　　AKA的发展出谋划策，提出了许多有重要价值和意义的建议和思想。

　　　　最后根据大家的讨论，AKA决定将下一阶段的工作重点放在下面几个方面：
　　* 一是要在自由软件文化的宣传、推广和资料整理方面多做工作；
　　* 二是要充分发挥现有AKA组织中软件工程人才强的优势，以软件工程作为AKA具体工
　　作的突破口，筹办AKA软工中文网站，推广软件工程的思想，在今后工作中逐步总结形
　　成具有自己特色的软件开发模式；
　　* 三是要与北京Lisoleg Linux源码学习小组共同推动Linux核心在中国计算机人才中
　　的推广工作，并打算举办这方面的讲座；
　　* 四要逐步丰富，美化AKA的WWW网站，将www.aka.citf.net办成一个更有影响力的技
　　术站点；
　　* 五是要加强与企业和媒体的协作，宣传和推动AKA发展；
　　* 六是要加强AKA组织工作，多学习国外组织发展经验，同时也要注重AKA新成员的发
　　现和培养工作。

　　　　这次沙龙对增进AKA凝聚力，指导AKA下一阶段发展起到了积极的促进作用。
