2000 Mar 25

　　第一次讲座通知：

　　主题： Linux 编程入门以及基本编程工具的使用
　　时间： 4 月 2 日（星期日）下午 2:00
　　地点： 清华大学西主楼 2 区 404 （从西主楼二区上四楼，一直右转，
　　　　见到“A 管理模式培训中心”即是）。

　　内容简介： 俗话说：“工欲行其事，必先利其器”。系列讲座的第一
　　讲，就从 Linux 上常用的编程工具开始讲起。在讲解这些工具之前，计
　　划简要讲解 Linux 系统 的发展历史和基本特色，这部分将对比大家熟
　　悉的操作系统讲解。通过这些介 绍使大家对 Linux 有一个较为全面的
　　了解，然后在此基础上讲解最常用的编程 工具。

　　本次讲座主要包括如下内容：
　　　　* 开场白
　　　　* Linux 的历史以及相关人物和事件
　　　　* Linux 与常见操作系统的区别
　　　　* Linux 上开发程序的典型场景（演示）
　　　　* C 程序的编译联结过程
　　　　* 函数库和系统调用的关系
　　　　* Shell 脚本编程（实例）
　　　　* vi、emacs、gcc/egcs、gdb、make 等常用工具（实例）

2000 Mar 25　

　　　　　　　　　　 Linux编程系列讲座
　　Linux下的编程！！！
　　哇，我也能成为人所仰慕的编程高手吗？

　　是的，你能！只要你现在起开始行动。

　　也许，你迈向大师的道路就从AKA的讲座开始......

　　欢迎你报名参加“Linux编程系列讲座”。
　　=====================================================================
　　一、讲座形式
　　　　免费讲座，提前报名，自由听讲，计算机投影演示，可安排会后小组讨
　　论， 并可在AKA的BBS上讨论等。
　　　　由MiniGUI的开发者kongming主讲，并将邀请其他高手。

　　二、预备知识
　　1、首先，你应该有C语言的基础
　　2、最好还有一些 Linux 的使用经验
　　3、最好具备一定的编程基础

　　三、讲座内容（可能会根据实际需要做一些变动）
　　1. Linux 编程入门以及基本编程工具的使用（编译器与联结器、函数库和系统
　　　 调用，Shell 脚本，vi、emacs、gcc/egcs、gdb、make）
　　2. 自由软件项目的组织、管理和发布（autoconf、automake、
　　　 diff/patch 及 CVS，RPM 包管理系统）
　　3. 其他 Linux 工具及基础性知识（其他工具及语言简介）
　　4. Linux 系统调用介绍及与其他 UNIX 系统之间的区别
　　5. Linux Kernel 的结构和编译（kernel 体系结构，重要功能特色，内核配置
　　　 及编译）
　　6. Linux 驱动程序结构
　　7. Linux 编程主题一——文件系统及 I/O（文件、目录、基本 I/O、标准 I/O）
　　8. Linux 编程主题二——进程和进程间通讯（进程，进程控制、进程关系，
　　　 管道、FIFO、信号灯、共享内存和消息队列）
　　9. Linux 编程主题三——信号
　 10. Linux 编程主题四——终端和控制台（终端编程，终端本地化，伪终端，
　　　 控制台图形编程）
　 11. Linux 编程主题五——Perl 语言
　 12. X 编程主题一——（QT 部分）
　 13. X 编程主题二——（GTK 部分）
　 14. I18N 机制说明（KDE 的本地化方法）
　 15. Linux 的网络特色和配置
　 16. BSD Socket 编程
　 17. MySQL 及 PostgreSQL 数据库介绍及对比（SQL 语言，MySQL、PostgreSQL
　　　 介绍及对比，Linux 下的 C 语言数据库接口）
　 18. Apache Web 服务器相关内容（配置、cgi 脚本概念及实例，PHP3 编程）
　 19. CCE 项目介绍和 ZWinPro 项目介绍
　 20. MiniGUI 项目介绍
　 21. SmartBootManager 项目介绍
　 22. LIDS 项目介绍

　　四、时间和地点
　　　　地点：清华大学西主楼2区404（从西主楼二区上四楼，一直右转，见到
